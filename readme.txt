Redirect Plugin is 301 redirect plugin for wordpress site. It help cloak links, short links and track click count.

== Description ==

Redirect Plugin have unique feature that help you to stop voilation of link redirect for e-commorse website like amazon, ebay.Now use redirect theme while link is being redirecting.

== Installation ==
1. Upload `redirect-plugin.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can use RedirectPlugin

== Frequently asked questions ==

* Why we get 404 error when go to short links ?
Short links are using a custom post so it change permalink structure. Simply you need to save your permalink setting and start using plugin. Go to Setting->Permalink & Click on save button.
